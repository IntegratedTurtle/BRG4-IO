# Was ist ein Club?
Ein Club ist eine Versammlung von SchülerInnen der Oberstufe, die sich
regelmäßig treffen und dabei einem gemeinsamen Hobby oder einem Interesse
nachgehen. (z.B. Schach-, Debattier-Club, Musik-Trio, …)
